React 的基础语法

JSX ​

JSX 是 JavaScript 的语法扩展, 允许我们在 JavaScript 文件中写类似 HTML 的标记. 最终会被编译成 React.createElement 调用

tsx
const App = () => {
  return (
    <div>
      <h1>Hello, World!</h1>
      <p>This is a simple paragraph.</p>
    </div>
  )
}

export default App

HTML 标签 ​

JSX 的标签写法比传统 HTML 更严格

  1. 必须有且只有一个根元素
tsx
<h1>Hello, World!</h1>
<p>This is a simple paragraph.</p>
tsx
<div>
  <h1>Hello, World!</h1>
  <p>This is a simple paragraph.</p>
</div>
  1. 如果不想渲染多余的 DOM 节点，可以使用 Fragment (空标签)
tsx
<>
  <h1>Hello, World!</h1>
  <p>This is a simple paragraph.</p>
</>
  1. 标签必须严格闭合

所有标签都必须显式闭合. 自闭合标签必须加 /

tsx
<img src="" alt="">
tsx
<img src="" alt="" />
  1. 属性命名采用驼峰式

JSX 更接近 JavaScript 而非 HTML, 所以大部分属性名需变更

  • class -> className
  • onclick -> onClick
tsx
<div className="container" onClick={handleClick}/>

JavaScript 表达式 ​

插入 JavaScript 表达式, 使用花括号 {}

插值写法

html
<div>
  <h1>Name: {name}</h1>
  <p>Age: {age}</p>
</div>
ts
const name = 'XYZ'
const age = 25

属性写法

html
<img src={imgSrc} alt="" />
ts
const imgSrc = 'example.jpg'

注意

{} 内不能放 if 语句或 for 循环, 只能放表达式 (有返回值的)

内联样式 style 的值必须使用对象 (且属性名驼峰), 而不是 CSS 字符串

tsx
<div style={{ backgroundColor: 'black', color: 'white' }} />
tsx
const divStyle = { backgroundColor: 'black', color: 'white' }
return <div style={divStyle} />

条件渲染 ​

三元表达式写法

tsx
{isLogin ? <span>Welcome</span> : <button>Go Login</button>}

逻辑与写法

tsx
{isLogin && <span>Welcome</span>}

列表渲染 ​

使用 map

html
<ul>
  {list.map((item, index) => (
    <li key={item.id || index}>{item.name}</li>
  ))}
</ul>
ts
const list = [
  { id: 1, name: 'Apple' },
  { id: 2, name: 'Banana' },
  { id: 3, name: 'Orange' },
]

优先使用数据自带的 id, 万不得已才用 index

注释 ​

在 JSX 中, 注释也必须写在 {} 表达式内部, 否则会被当成文本节点输出

tsx
{/* This is a simple paragraph. */}
