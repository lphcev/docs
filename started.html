本地环境搭建 ​

从 Github 克隆

bash
> git clone https://github.com/lphcev/raxly-document.git

到项目根目录

bash
> cd raxly-document

安装依赖

bash
> npm i
bash
> pnpm i

TIP

这个项目默认使用 pnpm 作为包管理器, 同时也兼容了 npm, 可以自由切换

如果想切换成 npm, 需要删除项目下的 pnpm-lock.yaml 和 pnpm-workspace.yaml 并重新安装依赖

启动命令 ​

bash
> npm run docs:dev
bash
> pnpm run docs:dev

WARN

使用 pnpm 可以正常启动, 但是结束之后控制台会有以下报错

[ELIFECYCLE] Command failed with exit code 1.`

这属于正常现象, 不会影响使用. 如果使用 npm 就不会有这个报错

构建命令 ​

普通构建

bash
> npm run docs:build
bash
> pnpm run docs:build

构建时指定 base 路径

bash
> npm exec -- vitepress build docs --base /docs/
bash
> pnpm exec vitepress build docs --base /docs/

执行成功之后会生成 /raxly-document/docs/.vitepress/dist 目录
