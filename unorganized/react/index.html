React 知识点总结

环境 React 19 + Typescript 6

理解 React ​

React 是一个用于构建用户界面的 JavaScript 库. 以下介绍 React 的核心特性

组件化 ​

React 可以把页面拆分成一个个独立, 可复用的组件, 每个组件只负责自己的逻辑和外观, 像搭积木一样拼成完整页面, 这让代码更容易编写, 测试和复用

tsx
const Message = ({ text }: Props) => {
  const style = { color: 'red' }
  return <div style={style}>{text}</div>
}

const App = () => {
  return <Message text="Hello, World!" />
}

声明式编程 ​

声明式编程是一种编程范式, 核心思想是: 你只关心 "想要什么结果", 而不关心 "如何一步步实现"

与命令式编程比较

名称关注点
命令式编程如何做: 写出每一步操作
声明式编程要什么: 根据数据描述最终界面
html
<button id="btn">Count: 0</button>

<script>
  let count = 0
  const btn = document.getElementById('btn')

  btn.addEventListener('click', () => {
    count++
    btn.textContent = `Count: ${count}`
  })
</script>
tsx
import { useState } from 'react'

const App = () => {
  const [count, setCount] = useState(0)
  return <button onClick={() => setCount(count + 1)}>Count: {count}</button>
}
