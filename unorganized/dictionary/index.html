Python 脚本 ​

快速定位到本地自动化部署 py 位置

bash
> D:
> cd D:\Development\raxly-automation\src\deploy

部署自动化脚本

bash
> python rx_automation.py

部署文档

bash
> python rx_document.py

部署简历

bash

Git 首次建仓提交 (Raxly 系列产品)

bash
> git add *
> git commit -m "BASE: Project Initialization"
> git tag v0.0.0
> git checkout -b dev
> git push --all origin
> git push origin --tags

Git 之后的修改

bash
> git add *
> git commit -m "."
> git push --all origin

Git 设置上游分支

bash
> git branch --set-upstream-to=origin/dev
